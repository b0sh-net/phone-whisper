## Recommended post

I built an Android dictation app because keyboard dictation kept missing for me, and Gemini's voice input didn't fit the "dictate, then keep editing the draft" workflow I wanted.

So I made Phone Whisper: a push-to-talk overlay that inserts text into the app you're already using.

- works with SwiftKey/Gboard
- local or cloud transcription
- optional cleanup/post-processing
- open source

Repo: https://github.com/kafkasl/phone-whisper
APK: https://github.com/kafkasl/phone-whisper/releases
